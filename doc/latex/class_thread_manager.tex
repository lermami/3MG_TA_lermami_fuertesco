\doxysubsection{Detailed Description}
This class manages a pool of threads that can be used to execute tasks concurrently. 

The \doxylink{class_thread_manager}{Thread\+Manager} class provides functionality to add tasks to a queue, spawn worker threads, and manage their execution. It allows adding functions as tasks, checking the status of futures, and waiting for them to complete.
