\doxysubsection{Detailed Description}
This class manages the loading and access of various resources for the engine. 

The \doxylink{class_resource_manager}{Resource\+Manager} class provides a central system for loading textures, geometries, vertex buffers, and index buffers. It allows asynchronous loading, waiting for resources to be ready, and retrieval of resources by name.
