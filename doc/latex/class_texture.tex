\doxysubsection{Detailed Description}
This class represents a texture object used for rendering. 

The \doxylink{class_texture}{Texture} class provides functionality to load and manage textures for use in rendering pipelines. It allows loading textures from files or creating them in memory, and setting various texture parameters like wrapping mode and filtering.
