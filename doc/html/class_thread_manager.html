Detailed Description

This class manages a pool of threads that can be used to execute tasks concurrently.

The ThreadManager class provides functionality to add tasks to a queue, spawn worker threads, and manage their execution. It allows adding functions as tasks, checking the status of futures, and waiting for them to complete.

Template method to add a task to the queue and return a future object.
