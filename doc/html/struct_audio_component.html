Detailed Description

Struct representing an audio component attached to an entity.

This struct manages a sound source object and allows attaching a sound buffer to it for playback.

Constructor that creates an AudioComponent with a sound source and optionally plays it.

This constructor takes various arguments to initialize the sound source:

Parameters
buffReference to a SoundBuffer object containing the audio data.
nameA name for the sound source (for debugging purposes).
posPosition of the sound source in 3D space (ALfloat array of length 3).
speedVelocity of the sound source (ALfloat array of length 3).
gainGain of the sound source (volume amplification). Defaults to 1.0f.
pitchPitch of the sound source (alters playback speed). Defaults to 1.0f.
playIf true, starts playing the sound after creation. Defaults to false.

After setting up the sound source, it adds the provided sound buffer using the addSound function and optionally starts playback using the Play function.
